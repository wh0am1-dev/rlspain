## :computer: how-to

:hammer: **`npm start`**<br>

:triangular_ruler: **`npm run gen`**<br>

:rocket: **`npm run deploy`**<br>

lint and validate deps

:mag: **`npm run inspect`**<br>
visualise the app structure
